# campo_texto

## Apenas para desenvolvimento de campos de entrada 

### Como emular?
<br />
 Seguindo os passos, primeiramente instalando os pacotes
<br />
<code> flutter pub get </code>
<br />
Com o emulador e/ ou  celular conectado:
<br />
<code> flutter run </code>
